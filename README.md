# Obscion for Zed

A collection of 10 carefully crafted themes for the Zed editor, featuring warm and vibrant color palettes inspired by obsidian.

### Dark Themes
- **Obscion Dark** - The original theme with warm, vibrant colors on a dark background
- **Obscion Midnight** - Ultra-dark variant with deeper blacks and higher contrast
- **Obscion Ocean** - Blue-shifted dark theme with cooler tones
- **Obscion Forest** - Green-shifted dark theme with natural, earthy vibes
- **Obscion Sunset** - Orange and red heavy theme with warm, fiery colors
- **Obscion Monochrome** - Desaturated dark theme with minimal color distractions
- **Obscion Amber** - High contrast warm theme with golden tones

### Light Themes
- **Obscion Light** - Clean light theme with darker, saturated accent colors
- **Obscion Dawn** - Soft, warm light theme with beige and cream tones
- **Obscion Lavender** - Purple-shifted light theme with elegant, soft colors

## üì¶ Installation

### From Zed Extensions (Recommended)
1. Open Zed
2. Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows/Linux)
3. Type "extensions" and select "zed: extensions"
4. Search for "Obscion"
5. Click "Install"

### Manual Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/heyobscion/Obscion-for-Zed
   ```
2. Copy the entire folder to your Zed extensions directory:
   - **macOS**: `~/.config/zed/extensions/`
   - **Linux**: `~/.config/zed/extensions/`
   - **Windows**: `%APPDATA%\Zed\extensions\`

3. Restart Zed

## üöÄ Usage

1. Open Zed settings (`Cmd+,` on Mac, `Ctrl+,` on Windows/Linux)
2. Search for "theme"
3. Select your preferred Obscion variant from the dropdown

Or use the command palette:
1. Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows/Linux)
2. Type "theme" and select "zed: select theme"
3. Choose your favorite Obscion variant
